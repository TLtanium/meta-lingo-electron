## 六、差异原因深度分析

### 1. 词典系统差异

**根本原因**: PyMUSAS 和 Wmatrix 使用不同的词典系统。

- **Wmatrix**: 使用 UCREL 开发的原始 USAS 词典，经过多年学术积累
- **PyMUSAS**: 使用 `en_dual_none_contextual` 模型，词典覆盖范围和优先级不同

**具体表现**:
- 技术术语覆盖不同（如 `firewall`, `co2`）
- 领域专有词汇处理不同（如 `jaguars` 作为汽车品牌 vs 动物）

### 2. 未知词处理策略差异

**Wmatrix**: 对无法识别的词使用 `Z99`（未知）标签

**应用 (PyMUSAS)**: 更倾向于基于形态和上下文给出具体标签
- Wmatrix 的 Z99 词在应用中被标注为 T1.3 (12次), Z2 (7次), Z1 (6次) 等

### 3. 消歧算法差异

**Wmatrix 消歧**: 基于 UCREL 特有规则，结合语料库统计信息

**应用消歧**: 基于 PyMUSAS + 自定义消歧逻辑
- 话语域识别（统计 21 大类占比）
- 一文一义启发式（同文章同词同标签）
- 文本类型优先级

**结果**: 对多义词的处理策略不同，导致标签选择差异
- 如 `fast`: Wmatrix=Z3 (虚词), App=N3.8+ (速度快)/F1- (禁食)

### 4. MWE (多词表达) 处理差异

**Wmatrix**: 有完善的 MWE 词典，为多词表达添加 `_MWE` 后缀

**应用**: MWE 检测依赖 PyMUSAS 的 `pymusas_mwe_indexes`
- 检测算法可能覆盖范围不同
- 某些 Wmatrix 识别的 MWE，应用可能未识别

### 5. 分词差异

**Wmatrix**: 使用 CLAWS (Constituent Likelihood Automatic Word-tagging System)

**应用**: 使用 SpaCy (`en_core_web_lg`)
- 分词边界可能不同
- 影响到标签分配的词语边界

### 6. 频率统计差异

**Wmatrix 总频率**: 121,004  
**应用总频率**: 50,962

**可能原因**:
- 标点符号处理方式不同
- 停用词过滤策略不同
- MWE 计数方式不同（整体 vs 拆分）

### 7. 标签粒度差异

- Wmatrix 使用更多强化标签（如 `A12++++++`, `N3.7+++`）
- 应用可能简化为基础标签
- 性别/类型标记差异（如 `mf`, `mfn`, `c` 后缀）

## 七、改进建议

### 短期优化
1. **完善 Z99 兜底处理**: 对 PyMUSAS 返回的部分明显不合理标签进行二次校验
2. **扩展 MWE 词典**: 参考 Wmatrix 的 MWE 列表补充缺失的多词表达
3. **添加领域词典**: 为技术文档、新闻等特定领域添加专有词汇

### 中期优化
1. **优化消歧算法**: 分析 Wmatrix 的消歧规则，调整应用的消歧优先级
2. **统一分词策略**: 考虑使用与 CLAWS 更接近的分词器
3. **添加上下文感知**: 利用 BERT 等模型增强消歧能力

### 长期优化
1. **训练自定义模型**: 基于 USAS 标注语料训练专用模型
2. **构建反馈系统**: 允许用户标注错误，持续改进词典
3. **多模型融合**: 结合多个标注器的结果进行投票

## 八、技术说明

### 对比方法
1. 解析 Wmatrix 和应用的输出文件
2. 标签标准化（移除 `_MWE` 后缀、性别标记等）
3. 按标签统计频率、按(词,标签)对计算匹配率
4. 分析混淆模式和差异原因

### 准确率计算
- **语义域覆盖率** = 共同标签的匹配频率 / Wmatrix 总频率
- **词语标注准确率** = 精确匹配(词,标签)对的频率 / Wmatrix 总频率

## 九、结论

### 准确率总结

1. **包含所有标签（含 Z 类）**: 
   - 语义域覆盖率: 33.13%
   - 词语标注准确率: 30.15%

2. **排除 Z 类语法标签后**:
   - 语义域覆盖率: 25.30%
   - 词语标注准确率: 22.66%

**观察**: 排除 Z 类后准确率下降，说明 Z 类标签（语法词、标点、未知词）在 Wmatrix 和应用中相对一致，而其他语义标签差异更大。这反映了两个系统在语义理解层面的根本差异。

### 主要差异来源

当前应用与 Wmatrix 的准确率约为 22-33%（取决于是否包含 Z 类），主要差异来源于：

1. **词典系统不同**: PyMUSAS vs UCREL USAS 词典
2. **未知词处理**: 应用更积极分配标签，Wmatrix 保守使用 Z99
3. **消歧策略差异**: 多义词的标签选择不同
4. **MWE 覆盖差异**: 多词表达识别范围不同

这些差异是两个独立开发的语义标注系统之间的固有差异，不完全代表标注质量问题。在实际应用中，用户应了解这些差异并根据研究需求选择合适的工具。
